To contact a collection directly, locate its Collection Profile:

| Step 1: Select "Search Collections" from your portal's main menu. |
| :-: |
| ![Locating contacts step 1](/img/contactssearch.png)|

| Step 2: Scroll down to the "Collections" section of the search form and select "more info...". |
| :-: |
| ![Locating contacts step 2](/img/contactsmoreinfo.png)|

| Step 3: Information for the person(s) responsible for the collection will be listed on it's profile.  |
| :-: |
| ![Locating contacts step 3](/img/contactsexample.png) |
